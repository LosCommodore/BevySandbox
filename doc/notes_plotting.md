Plotting Libraries

bevy_plot: A dedicated plotting library for Bevy with a focus on aesthetics and interactivity. It handles both data points (scatter/line) and explicit functions, which it renders using quadratic Bezier interpolation for smooth curves. It supports interactive features like zooming and panning. 

egui_plot: An immediate-mode 2D plotting library specifically for egui. This is a highly popular choice for developers already using bevy_egui for their UI, as it provides a simple widget-based API for line and scatter plots.
